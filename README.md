# Leon - Creative Agency Template

## 📖 About The Project

This is my first complete HTML/CSS/JS project – a responsive landing page for a creative agency called "Leon." I built this to apply everything I learned about modern CSS layouts, responsive design, and basic DOM manipulation.

**What makes this special:** I didn't just copy code. I built it from scratch, using a mobile-first approach, and added JavaScript to handle the navigation menu interactivity.

## 🛠️ Built With

- **HTML5** – Semantic elements (`<main>`, `<section>`, `<header>`, `<footer>`)
- **CSS3** – CSS Variables, Flexbox, CSS Grid, Media Queries
- **JavaScript** – DOM manipulation for mobile menu toggle
- **Font Awesome** – Icons
- **Google Fonts** – Work Sans font

## ✨ Key Features

- ✅ **Fully Responsive** – Works on mobile, tablet, and desktop
- ✅ **Mobile-First Design** – Built for small screens first, then enhanced
- ✅ **CSS Variables** – Easy theming (`--main-color`, `--section-pad`, etc.)
- ✅ **Interactive Menu** – Click the hamburger icon to toggle the navigation
- ✅ **Clean, Minimal Design** – Focus on content and usability
